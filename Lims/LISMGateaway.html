Utimaco's Lawful Interception Management System LIMS

Intercept Communications - Lawfully


Utimaco LIMS is a certified and industry-leading solution to help telecommunications service providers respond to electronic surveillance orders as required by law.

Real-time Call Conversion and Transcoding

Utimaco LIMS Gateway

The Utimaco LIMS Gateway is a powerful, yet cost-effective media gatewayappliance fully integrated with the Utimaco Lawful Interception Management System (Utimaco LIMS).

The LIMS Gateway converts intercepted calls between SS7/ISDN and VoIP networks. It supports all common voice codecs including transcoding and protects calls by comprehensive access control and encryption features.

In combination with Utimaco LIMS the gateway mediates intercepted calls in compliance with common LI standards, such as ETSI TS 102 232-5 and TS 101 671.

Features

PSTN and VoIP protocols:
  • SS7 ISUP signaling with several national variants
    (ITU, ANSI, Bellcore, UK, China, India and Russia)
  • ISDN (ITU, ETSI, ANSI)
  • SIP (RFC3261, RFC 3261), RTP
  • SIGTRAN M2UA (RFC 3331)
  • SCTP (RFC 2960)
  • Megaco/H248
Echo Cancellation:
  • G.168-2002 with 128ms tail
Transcoding
  • G.711
  • G.723.1
  • G.726
  • iLBC
  • G.729AB
  • GSM
  • G.722
  • AMR
  • G.722.1
  • T.38 Fax Relay
DTMF Detection and Generation:
  • RFC2833 Tone relay
  • In-band
  • DTMF detection and generation

Benefits

Costs

The Utimaco LIMS Gateway avoids costly network upgrades and changes at the LEMF by realtime conversion, mediation, buffering and transcoding of calls LIMS gateway is offered as a cost-efficient turn-key appliance

Flexible use

LIMS Gateway supports a large range of voice codecs and can be configured for multiple use cases and network environments LIMS gateway scales for hundreds and thousands of concurrent calls by adding E1/T1 trunk cards and appliances

Compliance

LIMS Gateway supports multiple signaling and media protocols including national variants. The handover interface is fully compliant to common ETSI standards
